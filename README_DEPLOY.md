# Docenty-fit Portfolio Website

정적 HTML 단일 페이지 사이트입니다. Vercel에서 바로 배포할 수 있습니다.

## Vercel 배포
1. 이 `site` 폴더를 GitHub 저장소에 업로드합니다.
2. Vercel → New Project → 해당 저장소 Import.
3. Framework Preset은 `Other` 또는 Static로 두고 Deploy.
4. 배포 후 생성된 URL을 이력서/PPT에 추가합니다.

## 파일 구조
- `index.html`: 도슨티 맞춤형 포트폴리오 + Prompt MVP Builder 데모
- `README_DEPLOY.md`: 배포 안내

## 참고
현재 버전은 외부 API 없이 브라우저에서 동작하는 MVP 데모입니다.
실제 AI 생성/로그인/결제 기능은 추후 Supabase + Vercel Functions + OpenAI API + Toss/Stripe로 연결하면 됩니다.
